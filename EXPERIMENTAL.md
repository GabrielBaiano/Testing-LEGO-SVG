### Notas de Implementação
- **LEGO Autêntico**: Cada peça agora está envolta em um link `<a>`. Esse comportamento provou ser o mais estável para remover espaços entre blocos no GitHub.
- **Preenchimento Lateral**: Aumentei para **12 blocos por linha** (1200px) para garantir que o fundo cinza ocupe toda a largura disponível na maioria dos monitores.
- **Zero Gaps**: Todas as tags estão coladas no código fonte.
