### Notas de Implementação
- **Barras Largas (Wide Bars)**: Seguindo o feedback, substituímos os muitos quadradinhos por barras largas "prontas" (1200px para o fundo e 200px para as laterais).
- **Estabilidade**: Menos tags `<a>` e `<img>` significam menos pontos onde o GitHub pode inserir espaços indesejados.
- **Preenchimento Lateral**: O uso de barras de 200px nas laterais garante que o conteúdo centralizado de 800px pareça estar "dentro" de uma tela infinita de cinza.
